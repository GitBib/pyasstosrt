    Subtitle¶

    class pyasstosrt.Subtitle(filepath: str | PathLike, removing_effects: bool = False, remove_duplicates: bool = False)¶

    Bases: object

    Converting ASS (Advanced SubStation Alpha) subtitles to SRT format.

    This class provides functionality to read an ASS subtitle file, convert its contents to SRT format, and export the result either as a file or as a list of dialogues.

    Parameters:¶
    filepath : Path¶

    Path to a file that contains text in Advanced SubStation Alpha format

    removing_effects : bool¶

    Whether to remove effects from the text

    remove_duplicates : bool¶

    Whether to remove and merge consecutive duplicate dialogues

    Raises:¶

    FileNotFoundError – If the specified file does not exist

    Variables:¶
    filepath: str | PathLike¶

    The path to the input ASS file

    file

    The stem (filename without extension) of the input file

    raw_text

    The raw content of the input file

    dialogues

    List of Dialogue objects representing the subtitles

    removing_effects: bool = False¶

    Flag indicating whether to remove effects from the text

    is_remove_duplicates

    Flag indicating whether to remove and merge consecutive duplicate dialogues

    Example:¶

    >>> from pyasstosrt import Subtitle
>>> sub = Subtitle("path/to/subtitle.ass", removing_effects=True, remove_duplicates=True)
>>> sub.convert()
>>> sub.export("output/directory", encoding="utf-8")

    Examples

    Basic usage:

    from pyasstosrt import Subtitle

# Convert a single file
sub = Subtitle('subtitle.ass')
sub.export()

# Convert with effects removal
sub = Subtitle('subtitle.ass', removing_effects=True)
sub.export()

# Convert with duplicate removal
sub = Subtitle('subtitle.ass', remove_duplicates=True)
sub.export()

# Convert with custom output directory and encoding
sub = Subtitle('subtitle.ass')
sub.export(output_dir='output', encoding='utf-8')

# Get dialogues without exporting
sub = Subtitle('subtitle.ass')
dialogues = sub.export(output_dialogues=True)

    export(output_dir: str | PathLike | None = None, encoding: str = 'utf8', output_dialogues: bool = False) → list[Dialogue] | None¶

    Export the subtitles either to a file or as a list of dialogues.

    If output_dialogues is False, this method exports the subtitles to an SRT file. Otherwise, it returns a list of Dialogue objects.

    Parameters:¶
    output_dir : Optional[Union[str, os.PathLike]]¶

    Export path for the SRT file (optional)

    encoding : str¶

    Encoding to use when saving the file (default is UTF-8)

    output_dialogues : bool¶

    Whether to return a list of dialogues instead of creating an SRT file

    Returns:¶

    List of Dialogue objects if output_dialogues is True, otherwise None

    Return type:¶

    Examples¶

    Basic Usage¶

    from pyasstosrt import Subtitle

# Convert ASS/SSA file to SRT
sub = Subtitle(filepath='input.ass')
sub.convert()
sub.export()

# Access dialogues
for dialogue in sub.dialogues:
    print(f"{dialogue.start} -> {dialogue.end}: {dialogue.text}")

    Advanced Usage¶

    from pyasstosrt import Subtitle

# Convert with options
sub = Subtitle(
    filepath='input.ass',
    removing_effects=True,
    remove_duplicates=True
)
sub.convert()
sub.export('output/directory', encoding='utf-8')
