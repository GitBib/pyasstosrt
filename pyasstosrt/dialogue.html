    Dialogue¶

    class pyasstosrt.Dialogue(index: int, start: str, end: str, text: str)¶

    Bases: object

    Represents a dialogue entry in a subtitle file.

    This class encapsulates a single dialogue entry, including its index, start and end times, and text content.

    Parameters:¶
    index : int¶

    The position of the dialogue in the subtitle file

    start: str¶

    The start time of the dialogue

    end: str¶

    The end time of the dialogue

    text : str¶

    The text content of the dialogue

    Examples

    Basic usage:

    from pyasstosrt import Dialogue, Time

# Create a dialogue entry
dialogue = Dialogue(
    index=1,
    start="0:00:10.00",
    end="0:00:15.00",
    text="Hello, world!"
)

# Get timestamp
timestamp = dialogue.get_timestamp()
print(timestamp)  # 00:00:10,000 --> 00:00:15,000

# Convert to string (SRT format)
srt_format = str(dialogue)
print(srt_format)
# 1
# 00:00:10,000 --> 00:00:15,000
# Hello, world!
